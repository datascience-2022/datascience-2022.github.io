Project for Data Science class.

This project aims to explore data science concepts learned during the course in practical application against song lyrics.

The project is divided into 2 main parts.

Part 1 is web scraping public sites to collect enough lyrics for analysis, including recursing through the site links to reach the required content and parsing HTML to extract the necessary data.

Part 2 is text analysis using NLTK library, this aims to build a model that analyses a song's lyrics and predicts the song's genre.

As we're not sure on the legal part of distributing scraped data - the lyrics file is encrypted, you'll have to run the web scraper part yourself if you want to build it up. The password will be supplied to the examinators.

Please use the .html link if you want to open the notebooks in view-only mode or use the .ipynb links if you would like to download the python notebooks to run them yourselves.
